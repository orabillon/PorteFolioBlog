# Blog / Porte folio
Projet passerelle 2

Le but de ce projet est de réaliser un site mettant à disposition un blog et un porte folio.


## On retrouve les fonctionnalités suivantes :

- un système de connection / de session
- une gestion de rôles 
- la création / modification / suppression d'article de blog
- l'affichage d'article avec possibilité de trier par date / theme 
- la possibilité de laisser des commentaires sur les articles de blog / projet
- la possibilité de créer / modifier / supprimer des projets 

## Liste des technologies utilisées : 

- HTML 
- CSS
- Sass
- Bootstrap 
- PHP / MySQL
